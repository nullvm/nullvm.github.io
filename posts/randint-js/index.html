Generate Ranged Random Integer in Javascript

October 26, 2024 · 1 min · Deni Andrian Prayoga | Suggest Changes

Here I will show you a simple way to generate a ranged random integer number in javascript. Since javascript doesn’t have built in function to generate ranged integer number, we will create one.

function randInt(left, right) {
  return Math.floor(Math.random() * (right - left + 1)) + left;
}
